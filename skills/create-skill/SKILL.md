# スキル作成スキル

新しいスキルを作成するときに使う。`skill-creator` スキルを活用してスキルを作成する。

## 手順

### 1. 保存先を確認する

スキルの保存先をユーザーに確認する。ユーザーレベルには2つの保存先があるので、必ずいずれかをユーザーに選ばせる:

- **ユーザーレベル** (`~/.agents/skills/`) — すべてのプロジェクトで使えるスキル。ローカル環境にのみ置く
- **プロジェクトレベル** (`./.agents/skills/`) — このプロジェクト専用のスキル

ユーザーレベルとして「~/.agents/skills/」「marketplace 配下」のどちらを指定されたか曖昧な場合は、勝手に決めず必ず確認する。

### 2. skill-creator を起動する

保存先が決まったら、`skill-creator` スキルを使ってスキルを作成する。その際、以下を skill-creator に伝える:

- スキルの目的・やりたいこと（ユーザーから聞いた内容）
- 保存先のパス（ステップ1で確認したもの）
- 日本語でスキルを作成すること
